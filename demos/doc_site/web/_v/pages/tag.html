在 Controller 中定义 _TAG_xxx 方法实现 TagLib 的功能，在 View 文件中使用 <包名.类名:Tag方法名> 调用一个 TagLib

如果在根控制器 index 中定义 Tag，调用时省略冒号前面的部分, 如 <:list></:list>

在 Controller 中的 Tag 处理函数

class c_bb_aa { function _TAG_list() { return [aa=>11,bb=>22,'list'=>[a=>1,b=>2,c=>3,d=>4]]; } }

调用 TagLib 的 View 代码

<bb.aa:list> aa:{$aa}<br/> bb:{$bb}<br/> <ul> <!-- #list --> <li> *** {$key}: {$value}</li> <!-- /list --> </ul> </bb.aa:list>

运行结果

aa:{$aa}
bb:{$bb}
*** {$key}: {$value}
